公司故事

有灵魂的游戏世界

「灵境」 — "灵"代表灵魂、智能；"境"代表游戏世界、虚拟空间。 合起来即"有灵魂的游戏世界"，完美契合 AI NPC 的核心理念。

同时，"灵境"是钱学森先生对 Virtual Reality 的经典翻译，承载着中国科技创新的传承意味。 我们相信，AI 不是替代游戏开发者，而是让每个人成为 10x 开发者。

从 creator_ai MCP 插件开始，我们正在构建完整的 AI 游戏开发工具链： 让 AI Agent 直接操控 Cocos Creator 引擎，让 NPC 拥有真正的灵魂， 让多 Agent 协作成为游戏研发的新范式。
