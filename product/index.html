creator_ai · 引擎层

MCP 工具 — 底层能力之一

加速内容生产与引擎集成，服务于 soul_npc 运行时部署。

零配置启动

安装插件后，通过 Creator 菜单启动 MCP 服务，本地 SSE 端点即可连接。

全场景覆盖

场景编辑、Prefab 模式、资源管理、脚本开发 — 完整工作流。

DSL 驱动

JSON DSL 描述 UI 结构，Bundle 导入即生成 Prefab。

安全可控

白名单字段、dryRun 预览与操作日志，自动化改动可审计。

快速开始

三步接入引擎工具链

01

安装插件

将 creator_ai 放入项目 extensions 目录。

client/extensions/creator_ai/
02

启动服务

在 Creator 编辑器中启动 MCP 服务。

AI Tools → MCP 服务 → 启动
03

连接 Agent

配置 MCP 客户端指向本地 SSE 端点。

http://127.0.0.1:3000/sse
