creator_ai · 灵境引擎

Cocos Creator
引擎桥接层

基于 MCP（Model Context Protocol）的编辑器插件。 将场景、资源、脚本操作封装为 16 个标准化工具，接入现有开发流程。

GitHub 仓库 技术文档

零配置启动

安装插件后，通过 Creator 菜单启动 MCP 服务，本地 SSE 端点即可连接。

全场景覆盖

场景编辑、Prefab 模式、资源管理、脚本开发 — 完整工作流。

DSL 驱动

JSON DSL 描述 UI 结构，Bundle 导入即生成 Prefab，适合批量界面生产。

安全可控

白名单字段、dryRun 预览与操作日志，自动化改动可审计。

快速开始

三步完成接入

01

安装插件

将 creator_ai 放入项目 extensions 目录。

client/extensions/creator_ai/
02

启动服务

在 Creator 编辑器中启动 MCP 服务。

AI Tools → MCP 服务 → 启动
03

连接端点

配置开发工具连接本地 SSE 端点。

http://localhost:3105/sse
MCP 工具

16 个标准化工具

覆盖基础、场景、资源、开发与诊断五类能力。

基础 · 2 场景 · 6 资源 · 3 开发 · 3 · 诊断 · 2
ping基础

健康检查，验证 MCP 连接。

get_project_info基础

获取项目名称、路径与引擎版本。

get_editor_context场景

获取编辑器场景上下文。

query_scene_hierarchy场景

查询节点树结构（深度上限 5）。

query_node_state场景

按 UUID 查询节点属性摘要。

apply_node_patch场景

save_editor_scene场景

保存当前场景或 Prefab。

toggle_project_preview场景
